# My Portfolio Backend

This is the backend for my personal portfolio.  
It handles API requests, authentication, and serves data for the frontend.

---

## 🚀 Tech Stack
- **Node.js**
- **Express.js**
- **MongoDB** (via Mongoose)
- **JWT Authentication**
- **CORS enabled**
